# NAME

comitup-web -- Wifi configuration web server for the Comitup service

## SYNOPSIS

`comitup-web`

## DESCRIPTION

__comitup-web__ is a web service that provides access to the comitup(8) Wifi
bootstrap service.

See comitup-conf(5).
